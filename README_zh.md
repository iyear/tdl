## 特性

- 单文件启动
- 低资源占用
- 吃满你的带宽
- 比官方客户端更快
- 支持从受保护的会话中下载文件
- 支持上传文件至 Telegram
- 导出历史消息/成员/订阅者数据至 JSON 文件
